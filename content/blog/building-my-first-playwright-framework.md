I'm a Manual Tester by day, and this project is how I'm building toward an SDET role.

## Why Playwright

Playwright gives reliable auto-waiting, strong TypeScript support, and a path into API testing with the same toolkit.

## Page Object Model

I structure pages as classes so locators and actions stay reusable. That mirrors how I'd want a team framework to grow.

## Current focus

I'm expanding coverage for login, registration, and job search flows on HiKode while keeping the repo readable for future CI.
